Main components of AI

  • Feature engineering : Is the process of identifying a minimal set of informative features from the provided dataset

  • Artificial Neural Network : It comprises of weighted interconnections between the set of computing nodes at consecutive layers .Most commonly used Artificial Neural Network are Convolutional Neural Networks , The robustness of convolution and Recurrent Neural Networks

  • Deep learning Architecture : It has more hidden layers between the input and output layers when compared to that of artificial neural networks
